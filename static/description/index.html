Tool Control

An employee used a tool but forgot to return or damaged it? No more lost inventory, no more lost profits! Tool Control ERP is a convenient repository for reports created in ToolControl mobile application.

The ToolControl app takes a photo of a worker at the moment when she/he takes a tool. Images are sent to Tool Control ERP, where you can always check who took this or that tool last and contact that person directly.
